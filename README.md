## 📋 Overview

**Sampath Food City Sales Analysis System** is a comprehensive desktop application developed for **Sampath Food City (PVT) Ltd**. The system automates the manual sales data analysis process, providing management with actionable insights for strategic decision-making.

> This application eliminates human bias, reduces time consumption, and provides reproducible, error-free sales analysis for better business decisions.

## ✨ Features

### 🔐 **Authentication System**
- Secure login with username, password, and role selection (Admin/User)
- New user registration with validation
- Password protection

### 📊 **Main Dashboard**
- Summary statistics cards (Total Revenue, Top Region, Best Product)
- Quick navigation to all analysis modules
- Recent activity feed

### 📈 **Best-Selling Products Analysis**
- Top products by quantity or revenue
- Horizontal bar chart visualization
- Sortable data table with rankings
- Date range filtering
- PDF export functionality

### 📉 **Product Performance Over Time**
- Individual product sales trends
- Line charts with date filters
- Daily/weekly/monthly views
- Key metrics panel (Peak sales, Average daily)
- Top region for selected product
- PDF export functionality

### 👥 **Customer Behavior Analysis**
| Analysis Type | Description | Visualization |
|--------------|-------------|---------------|
| **Day of Week Patterns** | Identify peak shopping days | Bar Chart + Pie Chart |
| **Monthly Sales Trends** | Seasonal pattern analysis | Line Chart + Bar Chart |
| **Popular Products** | Most purchased items | Bar Chart + Pie Chart |
| **Customer Purchase Frequency** | Loyal customer identification | Bar Chart |

- Dual chart visualization
- Detailed data tables
- Insight panels with recommendations
- PDF export functionality

### 🌍 **Regional Sales Analysis**
- Compare performance across branches
- Region-wise sales distribution
- Bar and pie chart visualizations
- Date range filtering
- Top region analysis
- PDF export functionality

### 👤 **User Management** (Admin Only)
- Add new users
- Update existing user details
- Delete users (with confirmation)
- Search users by username or role
- Role assignment (Admin/User)
- Table view of all users

### 📥 **Data Management** (Admin Only)
- CSV file import
- Data preview in JTable
- Load status monitoring
- Data validation
